# 开场欢迎语

环境检查通过后、进入 Phase 1 解析之前，**必须**先发这段欢迎语：

```
LinkedIn 寻访启动 — 三步走：

  Step 1 · 与你确认本轮寻访标准、建联话术
           ├ 拆解搜索策略
           └ 选定建联话术

  Step 2 · 自动跑 LinkedIn 搜索 + 记录详细 Profile 并筛选和评分
           └ 输出带优先级名单 → Excel 保存 + Dashboard 展示给你

  Step 3 · 你在 Dashboard 确认修改名单和话术 → 信息粘回这里
           └ 我帮你批量发 Connect

每步都会停下来等确认，不会偷跑。

接下来，把寻访需求发给我即可（公司、关键词、岗位、筛选偏好都行，格式随意）。
```

输出后等用户给输入，进入 Phase 1。
